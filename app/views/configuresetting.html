Manage Setting

Manage Auto Clear

When 'On' this removes records when an activity causes their quantity to reach '0'. Most users should leave this 'On'.

Allow Negative Quantity

Allow moving, tagging, converting, or updating units to result in negative quantities.

Default Quantity

If checked, all activity quantities will default to "1".(Quantity field is empty if unchecked).
