Default Settings

Manage Auto Clear

When 'On' this removes statements when an event causes their quantity to reach '0'. Most users should leave this 'On'.

Allow Negative Quantity

Allow moving, changing, converting, or updating units to result in negative quantities.

Default Quantity

If checked, all event quantities will default to "1".(Quantity field is empty if unchecked).

Make 'vertical entry' format to default

If checked, You will see the add Inventory form with vertical entry format.

Allow Socket Mobile scan

If checked, You will Scan barcode with socket mobile scanner.
